# Mighty Mouse

The main hardware components are:

* [ESP32 microcontroller](#esp)
* Motor Driver
* Voltage Regulator
* Motors
* Encoders
* Wheels
* Power Source


![Main Image](images/main.jpg)


##ESP32
ESP32 microcontroller is a low cost, low power system on a chip with integrated Wi-Fi.
